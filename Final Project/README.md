# Final Project - Arduino-Based Mechanical Ventilator
In this project, we decided to build an Arduino-based mechanical ventilator. We used the ESP32 microcontroller and Arduino Uno microcontroller alongside other sensors and actuators listed below to simulate Volume-Control Ventilation. We built a Flutter application to get patient data, and required Breaths Per Minute, and display sensor readings for monitoring purposes.

We used the following sensors and actuators:
- BMP280 Pressure Sensor
- AHT21B Temperature and Humidity Sensor
- KY039 Heartrate Sensor
- DC Motor
